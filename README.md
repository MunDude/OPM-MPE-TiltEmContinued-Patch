# OPM-MPE-TiltEmContinued-Patch
(THIS IS THE README FILE)
To Install:

1. Make sure you have in your GameData folder the following mods:

ModuleManager(version number, i.e 4.2.3).dll
Kopernicus & ModularFlightIntegrator
MPE
OPM

TiltEm (Make sure you downloaded the version from SpaceDock, it should be called "TiltEm Continued")

2. Take the "OPM_MPE__TiltemCompat.cfg" file out of the zip.

3. Drop the config file into your GameData folder.

4. Run the game.

5. Profit.

(If you encounter any errors or want to submit feedback send it on the forum page you downloaded this from.)
